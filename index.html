Summary: Latitude vs. X

The purpose of this project was analyze how weather changes as you get closer to the equator. The first step in this analysis was pulling data from the OpenWeatherMap API to assemble a dataset of over 500 cities.


After assembling the dataset, Matplotlib was used to plot various aspects comparing weather statistics in relation to latitude. Factors considered include: temperature, cloudiness, wind speed and humididty. This site provides the source data and visualizations created as part of the analysis. It also includes explanations and decriptions of trends and correlations witnessed.
